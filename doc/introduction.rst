Introduction
============

Welcome to the documentation of the Proxmox Automation Toolkit! This toolkit provides a set of Python scripts to automate the creation and provisioning of instances in Proxmox. The main components of the toolkit are the `packerbuilder.py` and `labbuilder.py` scripts. The former helps build Packer images, while the latter helps build a virtual lab in Proxmox.

This documentation will guide you through the installation process, basic usage, and advanced configurations for the toolkit. Whether you're a beginner or an experienced user, this documentation will provide the information you need to successfully use the Proxmox Automation Toolkit.
